# Privacy Policy

TensaLauncher connects to network services only for launcher features requested by the user or required for normal modpack operation.

## Network Services

The launcher may contact:

- Microsoft and Minecraft services for account authentication and Minecraft profile validation.
- Modrinth and CurseForge services for public modpack and mod metadata.
- TensaCraft services for TensaCraft modpack catalogs, update manifests, downloads, and user-initiated diagnostic reports.
- GitHub Releases for launcher update metadata and release assets.

## Diagnostic Reports

When a user chooses to submit a launcher report, the launcher sends diagnostic information needed to investigate the issue. Reports can include launcher logs, selected diagnostic text, platform information, and user-provided contact or description fields.

Users should remove passwords, tokens, personal data, and unrelated screenshots before submitting attachments.

## Local Data

The launcher stores configuration, profiles, logs, cache files, and installed modpack state on the user's machine. Local data is not uploaded unless the user explicitly submits a report or uses a feature that requires a network service.

## Disabling Network Features

Minecraft authentication, catalog downloads, update checks, and report submission require network access. Users can avoid report submission by not using the report action.
