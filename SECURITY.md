# Security Policy

## Supported Versions

TensaLauncher is supported through the latest stable release. Beta and prerelease builds receive fixes on a best-effort basis and should be updated to the latest available build before reporting a vulnerability.

## Reporting a Vulnerability

Do not open public issues for vulnerabilities, credential leaks, or exploitable bugs.

Use GitHub private vulnerability reporting when it is enabled for this repository. If private reporting is unavailable, contact the maintainers through the TensaCraft organization contact listed on GitHub.

Please include:

- affected launcher version and platform;
- steps to reproduce;
- expected and actual behavior;
- impact and whether user interaction is required;
- logs or screenshots with tokens, passwords, and personal data removed.

We will confirm the report after triage and publish a fixed launcher build when a code change is required.
